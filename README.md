# ankipo
This addon let's you add audio from (aws) polly to your deck.

[Ankiweb(2090845100)](https://ankiweb.net/shared/info/2090845100)

## config
```json
{
  "access_id": "",
  "access_key": "",
  "audio_field": "audio",
  "engine": "neural",
  "txt_field": "text",
  "region": "eu-west-1",
  "voice": ""
}
```
